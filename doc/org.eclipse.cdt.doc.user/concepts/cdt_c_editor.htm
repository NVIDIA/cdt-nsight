Source editor

Nsight provides an editor that gives you specific support for editing source code. This editor is invoked automatically when you edit a CUDA C/C++ source file.

The editor includes the following features:

  • Syntax highlighting
  • Content/code assist
  • Code folding
  • Integrated debugging features

You can customize some of the operation of the Editor view from the Window > Preferences > C/C++ > Editor preferences dialog.
